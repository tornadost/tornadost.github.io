📝 CyberBlogX: How to Publish a New Post

Overview

This guide explains how to add a new article to CyberBlogX by creating three files:

  • N.png – cover image
  • posts/feed/N/N.json – metadata & content
  • posts/feed/N/N.html – HTML loader

1. Add Cover Image

Place your image in:

/posts/assets/cover/

Name it exactly N.png, where N is the post ID (e.g. 3.png).

2. Create Post Folder & JSON

Inside /posts/feed/, create a folder named N. Then create N.json with:

{
  "title":   "My Awesome Idea",
  "excerpt": "A short teaser.",
  "date":    "2025-05-29",
  "read":    4,
  "tags":    ["tag1","tag2"],
  "content": "<p>Full HTML content here</p>"
}

Optionally, add "markdown": true and write full content in N.md.

3. Duplicate HTML Template

Copy posts/feed/1/1.html to posts/feed/N/N.html. The script will auto-detect N from the filename.

4. Verify & Publish

  1. Ensure JSON is valid (no trailing commas).
  2. Cover image path: /posts/assets/cover/N.png.
  3. Commit & push to GitHub Pages.

The home page will auto-fetch new JSONs and display the post card. Visiting /posts/feed/N/N.html loads your article.

Troubleshooting

  • Blank article: Check JSON syntax and that content or .md file exists.
  • Image 404: Confirm N.png name and path are correct.
  • Post missing on grid: Ensure N.json was created under /posts/feed/N/.
