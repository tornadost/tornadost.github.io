# Discord - widgetbot

## Example code:

```html
  <div class="container">
    <widgetbot
      server="1339396287061168232"
      channel="1339432343320137738"
      width="40vw"
      height="60vh"
    ></widgetbot>
    <script src="https://cdn.jsdelivr.net/npm/@widgetbot/html-embed"></script>
  </div>
```

> Check `discord.js` and `discord-raw.js` files for https://cdn.jsdelivr.net/npm/@widgetbot/html-embed

## The HTML code below can be pasted at the bottom of your website's body tag to embed Crate in your website:

```html
<script src='https://cdn.jsdelivr.net/npm/@widgetbot/crate@3' async defer>
    new Crate({
        server: '1339396287061168232', // Vidora
        channel: '1364145344526553170' // #staff
    })
</script>
```

## The HTML code below can be pasted where you want the widget to display to embed the widget in your website:

```html
<widgetbot
    server="1339396287061168232"
    channel="1364145344526553170"
    width="800"
    height="600"
></widgetbot>
<script src="https://cdn.jsdelivr.net/npm/@widgetbot/html-embed"></script>
```
