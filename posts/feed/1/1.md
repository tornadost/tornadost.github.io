# IRC Notifications to Mobile Device (HexChat)

A step-by-step guide to receive IRC mentions on your mobile device using HexChat and Pushover.

## Prerequisites

- A smartphone with Pushover installed  
- A Pushover account (user key & app token)  
- Windows PC with Python 3.8.10 and HexChat (matching bitness)

## 1. Install Pushover on Your Phone

1. Download **Pushover** from the App Store (iOS) or Play Store (Android).  
2. Sign up for an account and allow notifications when prompted.  
3. Log in at https://pushover.net/.

## 2. Install Python 3.8.10

1. Download **Python 3.8.10** for Windows 64-bit (or 32-bit if preferred) from the official site.  
2. Run the installer, **check “Add Python to PATH”**, and click **Install Now**.

## 3. Install HexChat

1. Download **HexChat** matching your Python bitness (64-bit or 32-bit).  
2. Run the installer and **enable** the **Python Language Interface** when selecting components.

## 4. Get the Python Notification Script

1. Click the link to the script (provided in original guide).  
2. Right-click the text, choose **Save As…**, and save it as `hexchat_pushover.py` in a temporary folder.

## 5. Configure Your Pushover Credentials

1. On Pushover.net, copy your **User Key** (top right of the page).  
2. Open `hexchat_pushover.py` in a text editor and set:
   ```python
   PUSHOVER_USER_TOKEN = 'your_user_key_here'
   ```
3. On Pushover.net, scroll to **Create an Application**, verify your email, name the app, and save.  
4. Copy the **API Token/Key** for this app and set in the script:
   ```python
   PUSHOVER_APP_TOKEN = 'your_app_token_here'
   ```
5. Save and close the file.

## 6. Load the Plugin in HexChat

1. Run HexChat.  
2. Go to **HexChat ▶ Plugins and Scripts ▶ Python Plugins**.  
3. Double-click `hexchat_pushover.py`.  
4. You’ll now receive Pushover alerts on your phone whenever your nickname is mentioned.

## 7. Autoload on Startup

To have the plugin load automatically:

1. Move `hexchat_pushover.py` to:
   ```
   C:\Users\<your_username>\AppData\Roaming\HexChat\addons\hexchat_pushover.py
   ```
2. Restart HexChat.
